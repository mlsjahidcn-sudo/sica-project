# Product Context: Study In China Academy (SICA)

## Why This Project Exists
International students face significant barriers when applying to Chinese universities:
- Fragmented information across hundreds of university websites
- Language barriers (most Chinese university sites are in Chinese)
- Complex application processes with varying requirements per university
- Limited guidance on scholarships and eligibility
- Difficulty tracking applications and documents

SICA solves these problems by providing a unified platform that guides students through the entire journey.

## How It Should Work

### Student Journey
1. **Discovery** → Browse universities and programs via searchable directory
2. **Assessment** → Take eligibility assessment to find matching programs
3. **Application** → Apply to programs, upload documents, track status
4. **Communication** → Use AI chat assistant for questions, get real-time notifications
5. **Result** → Receive admission decisions, manage enrollment

### Partner/Agent Journey
1. **Referral** → Refer students to the platform, track their progress
2. **Team Management** → Partner organizations can manage team members
3. **Task Management** → Assign and track tasks related to student applications
4. **Dashboard** → Analytics on referrals, applications, conversions

### Admin Journey
1. **Content Management** → CRUD for universities, programs, blog posts, success cases
2. **User Management** → Manage students, partners, admin users
3. **Application Review** → Review and process student applications
4. **Analytics** → Dashboard with metrics on platform usage

## User Experience Goals
- **Fast & Responsive** — Turbopack dev server, optimized production builds
- **Mobile-First** — Responsive design with shadcn/ui components
- **Accessible** — ARIA-compliant Radix UI primitives
- **Internationalized** — English and Chinese language support
- **Real-Time** — WebSocket-powered notifications for application updates
- **AI-Powered** — Chat assistant using Moonshot Kimi K2.5 for instant help
- **SEO Optimized** — Structured data, sitemaps, metadata for search visibility

## Key Differentiators
- All-in-one platform (not just directory or just application portal)
- AI chat assistant trained on Chinese university admissions
- Dual portal architecture (student + partner/agent)
- Comprehensive admin dashboard (not just basic CRUD)
- Real-time notifications across all user types
